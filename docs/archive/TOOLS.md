# 项目实施中使用的工具和方法

## 代码质量检查工具

### 1. Ruff
- **用途**：用于检查和修复 Python 代码的风格和语法错误
- **命令**：`ruff check .`（检查错误）、`ruff format .`（格式化代码）
- **配置**：在项目根目录下的 `pyproject.toml` 文件中配置
- **使用场景**：在实施过程中，我使用 Ruff 来检查和修复代码中的风格和语法错误

### 2. pytest
- **用途**：用于运行项目的测试套件
- **命令**：`python3 -m pytest tests/planner/ tests/integration/ -v`（运行特定模块的测试）、`python3 -m pytest tests/ -v`（运行所有测试）
- **配置**：在项目根目录下的 `pyproject.toml` 文件中配置
- **使用场景**：在实施过程中，我使用 pytest 来运行项目的测试套件，并修复了 Planner 模块的测试失败

## 代码调试工具

- **用途**：用于调试 Python 代码

- **使用场景**：在实施过程中，我使用 Python 调试器来调试 Planner 模块的测试失败
