# Nanobot 架构

## 架构文档

完整的架构文档请参考 [architecture/](./architecture/) 目录。

## 核心组件

### Main Agent
负责接收用户输入、调用子代理、生成最终响应。

### Sub Agent
专门处理特定任务的子代理系统。

### Prompt System V2
新的提示词系统，支持分层加载和钩子系统。

### Workflow
工作流编排系统，支持复杂任务的分解和执行。

## 架构特点

- **模块化设计**: 各组件独立可替换
- **插件化架构**: 支持通过插件扩展功能
- **配置驱动**: 通过配置文件控制行为
- **异步支持**: 全面的异步处理能力
